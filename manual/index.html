    Overview

    Installation

    Can be managed using yarn, npm, or jspm.

    yarn

    yarn add @combinatorics/set-partition

    npm

    npm install @combinatorics/set-partition --save

    jspm

    jspm install npm:@combinatorics/set-partition

    Usage

    :warning: The code requires regeneratorRuntime to be defined, for instance by importing regenerator-runtime/runtime.

    First, require the polyfill at the entry point of your application

    await import( 'regenerator-runtime/runtime.js' );
// or
import 'regenerator-runtime/runtime.js' ;

    Then, import the library where needed

    const setpartition = await import( '@combinatorics/set-partition' ) ;
// or
import * as setpartition from '@combinatorics/set-partition' ;
